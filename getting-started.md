# Getting Started

## Install with pip

```bash
pip install jupyterlab-a11y-checker
```

## Install via JupyterLab Extension Manager

1. Open JupyterLab.
2. In the left sidebar, open the Extension Manager.
3. Search for `jupyterlab-a11y-checker`.
4. Click Install and follow the prompts.
